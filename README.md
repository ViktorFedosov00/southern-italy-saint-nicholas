# По следам Святого Николая · Южная Италия

Сайт-репортаж о путешествии по югу Италии — **14–19 сентября 2026**.
Не рекламный лендинг, а гид-путешествие по местам: что там есть, что стоит
увидеть и почувствовать.

**Маршрут:** Бари → Полиньяно-а-Маре → Альберобелло → Матера.

🔗 **Живой сайт:** https://viktorfedosov00.github.io/southern-italy-saint-nicholas/

## Страницы

| Файл | Что внутри |
|------|------------|
| `index.html` | Главная — обзор путешествия, маршрут, карточки городов |
| `bari.html` | Бари: базилика и мощи Святого Николая, крипта, Бари Веккья, собор |
| `polignano.html` | Полиньяно-а-Маре: бухта Лама-Монакиле, белый город, гроты |
| `alberobello.html` | Альберобелло: трулли, Рионе-Монти, Трулло-Соврано (ЮНЕСКО) |
| `matera.html` | Матера: пещерные кварталы Сасси, рупестрианские церкви, закаты |
| `practice.html` | Кундалини-йога, ритм дня и раздел о Галине Гапон |
| `credits.html` | Источники фотографий — авторы и лицензии |

## О сайте

Статический сайт без сборки и зависимостей — чистый HTML/CSS/JS.
Общие стили в `assets/style.css`, скрипты (меню, появление при прокрутке,
лайтбокс) в `assets/app.js`. Фотографии — в `images/`.

**Фотографии** взяты из [Wikimedia Commons](https://commons.wikimedia.org)
под свободными лицензиями Creative Commons и отредактированы (масштабирование
и сжатие) для веба. Полный список авторов и лицензий — на странице
[`credits.html`](credits.html).

> ✏️ Фотографию и биографию Галины можно добавить на странице `practice.html`
> (раздел `#galina`). Контакт для записи вставляется в кнопки «Узнать о
> путешествии» — сейчас они ведут на блок «О путешествии» на главной.

— Путешествие с Галиной Гапон
